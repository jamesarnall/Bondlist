Bondlist

Sometimes it's useful to have a list of all the James Bonds.

Usage

Call https://jamesarnall.github.io/bondlist/bonds.json for a JSON data source.

The data looks like this:


        {
            "bonds": [
                {
                    "actor": "Sean Connery",
                    "id": "connery"
                },
                {
                    "actor": "David Niven",
                    "id": "niven"
